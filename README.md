# farmsavers


Developer information
=====================

Environment setup
-----------------

I'm using:
- Eclipse 4.4.0 (Luna)
- Say yes to all the plugins an m2 connectors to install
- Tomcat 7

To build with Maven
----------------------------

`mvn clean install`


Run the App
=================

Just bring up a browser and go to http://localhost:8080/farmsavers/app/index.html
